## 📌 Project Overview
This project analyzes the performance of the Namma Yatri platform. I transformed raw trip and search data into actionable insights using Power BI, focusing on the conversion funnel and supply-demand gaps.

## 🛠️ Tech Stack
* **Visualisation:** Power BI Desktop
* **Data Cleaning:** Power Query
* **Calculations:** DAX (Custom Measures)

## 📈 Key Insights
* **Conversion Funnel:** Analyzed the drop-off rates from 'Search' to 'Trip Completion'.
* **Revenue Analysis:** Tracked daily and hourly revenue patterns.
* **Driver Performance:** Visualized driver earnings and trip distances.
